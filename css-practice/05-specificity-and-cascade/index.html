CSS Specificity

To calculate specificity assign the value to each part of selectors.

  • Universal Selector: 0
  • Element Selectors and Pseudo Elements: 1
  • Class Selectors and Attribute Selectors: 10
  • Pseudo Class: 10
  • Id Selectors: 100
  • Inline Style: 1000

When specificity is the same, the browser uses the last declared rule


Cascade Algorithm

4 stages of the Cascade Algorithm

  1. Position and Order of Appearance: If two rules have the same specificity, the one written later win
  2. Specificity: The selector with higher specificity wins
  3. Origin: Determines which CSS rule wins based on where the style comes from (browser default CSS, your CSS, user stylesheet, or browser extension CSS)
  4. Importance (!important): This gets the highest priority among normal declarations.
